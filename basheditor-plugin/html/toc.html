Overview

Bash editor is just a bash editor plugin for eclipse

Features

  • Outline view
  • Quick-Outline dialog
  • Validation
  • Code assistence
  • Open referenced functions
  • Syntax highlighting
  • Colors can be customized. Also additional defaults for dark theme exists.
  • Bracket switching + highlighting
  • Block commenting (CTRL+7)
  • Auto bracket closing for [] and {}
  • Small footprint
